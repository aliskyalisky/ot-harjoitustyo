# Ohjelmistotekniikka, harjoitustyö
## Budjetointisovellus

Sovellus auttaa pitämään käyttäjä ajantasalla menoista ja tuloistaan.

### Asennus

1. Asenna riippuvuudet komennolla:

```bash
poetry install
```

2. Käynnistä sovellus komennolla:

```bash
poetry run invoke start
```
